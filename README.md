## Deployment
### Pre-requisits:
- Install Python3 (Python 3.7.3) with pip3
### Quick Setup
- Clone the repository: `git clone https://github.com/theinjector/amazonchecker`
- Install pip requirements: `pip3 install random`
### Run the program
- Usage: `python3 pwd.py`
